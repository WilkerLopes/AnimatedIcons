Começando

Os ícones que temos estão disponíveis para qualquer projeto que tecnologia WEB, usando JS e CSS. Perfeito para seu site ou WPA.

Para ter acesso a todos os ícones, siga os passos.

Github:

Começando na pasta do seu arquivo, estruture sua pasta para receber os arquivos. Modelo de estrutura básicas.

my_project
├── Plugins
│   └── AnimatedIcon
│       ├── css
│       │   └── animatedIcons.css
│       └── js
│           └── animatedIcons.js
└── index.html

Com os arquivos clonados em seu diretório, abra o arquivo que deseja usar os ícones e importe os arquivos, use tamnbém o Jquery, para que todos os códigos funcionem corretamente.

index.html
file_copy
  1. <html>
  2. <head>
  3. <title>my_project</title>
  4. <link rel="stylesheet" href="Plugins/AnimatedIcons/css/animatedIcons.css">
  5. </head>
  6. <body>
  7. <script src="Plugins/AnimatedIcons/js/animatedIcons.js"></script>
  8. <script src="https://code.jquery.com/jquery-3.5.0.min.js"></script>
  9. </body>
  10. </html>
